# hwvault-android

Android-specific components for hwvault (StrongBox, device integrations, approval UX).

## Current scope

- StrongBox-backed signing MVP scaffold for second-device approvals
- Canonical challenge payload format for approval signatures
- HTTP client contract compatible with hwvault resolver 2FA backend

See: `docs/STRONGBOX_MVP.md`
